## Methodology

### Ground Truth Source
- ClinicalTrials.gov API v2
- Author-reported results section
- Direct investigator submissions

### Matching Criteria
- Effect type match (case-insensitive)
- Value match within 2% tolerance
- CI match within 5% tolerance (when available)

### Deduplication
The extractor deduplicates effects with identical (type, value) pairs. This is correct behavior as:
1. Same effect reported across multiple CTG outcome rows should count as one detection
2. Prevents over-counting of redundant entries
3. Matches real-world usage where unique effects matter
